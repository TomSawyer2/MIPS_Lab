# 并行IO接口设计

## 运行

按照学习通4.1部分设计完硬件平台后，打开SDK直接新建模板，新建项目，新建C语言文件，并将对应代码直接复制进去，编译，烧录即可

**注意：学习通视频中GPIO_2没有开启中断，需要手动双击打开并将中断通过集线器连接到总线上**
